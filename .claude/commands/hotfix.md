# Comando /hotfix

Corrige un problema crítico con el menor riesgo posible.

## Qué hace

1. Lee `CLAUDE.md`
2. Identifica el problema urgente
3. Aplica una mitigación segura y acotada
4. Valida el sistema

## Reglas

- Priorizar estabilidad
- No ampliar alcance
- Documentación solo si el usuario la pide
