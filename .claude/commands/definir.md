# Comando /definir

Sirve para clarificar un pedido antes de implementarlo.

## Qué hace

1. Lee `CLAUDE.md`
2. Lee la conversación actual
3. Inspecciona el código relacionado
4. Devuelve user story, criterios de aceptación, supuestos y preguntas abiertas

## Reglas

- No implementa código
- No crea documentación salvo pedido explícito
