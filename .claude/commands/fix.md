# Comando /fix

Corrige un bug o comportamiento incorrecto.

## Qué hace

1. Lee `CLAUDE.md`
2. Reproduce o analiza el síntoma
3. Inspecciona el flujo real afectado
4. Aplica un fix de causa raíz cuando sea posible
5. Valida el resultado

## Reglas

- No documenta por defecto
- No hace refactors grandes no pedidos
