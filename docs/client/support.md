# Soporte y contacto

## Canales de soporte

### Soporte técnico
Para problemas con el sistema, errores o comportamientos inesperados:

- **Correo:** soporte@ejemplo.gob.ar
- **Horario de atención:** lunes a viernes de 8:00 a 18:00 hs

### Consultas funcionales
Para dudas sobre cómo usar el sistema o solicitar nuevas funcionalidades:

- Contactá al referente técnico de tu institución
- O escribí a soporte indicando que es una consulta funcional

## Cómo reportar un problema

Para que podamos ayudarte más rápido, incluí en tu mensaje:

1. **Qué estabas haciendo** cuando ocurrió el problema
2. **Qué esperabas que pasara**
3. **Qué pasó en realidad** (mensaje de error si aparece)
4. **Captura de pantalla** si es posible
5. **Fecha y hora** aproximada del problema

## Solicitar acceso o permisos

Los accesos al sistema son gestionados por el administrador de cada institución.
Si necesitás un usuario nuevo o cambios en permisos, contactá al administrador de tu institución.

## Reportar un problema de seguridad

Si encontrás una vulnerabilidad o acceso indebido a información, reportalo de forma urgente a soporte indicando en el asunto: **SEGURIDAD**.
