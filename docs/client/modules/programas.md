# Módulo: Programas

## ¿Qué es?

El módulo de Programas gestiona los programas sociales disponibles en la institución y las inscripciones de ciudadanos a cada uno.

## Funcionalidades principales

### Ver programas disponibles

Desde el menú lateral, ingresá a **Configuración → Programas** para ver el listado de programas activos en tu institución.

### Inscribir un ciudadano a un programa

1. Abrí el legajo del ciudadano
2. Ir a la pestaña **Programas**
3. Hacer clic en **Agregar programa**
4. Seleccionar el programa y completar los datos requeridos

### Seguimiento de inscripciones

Desde el detalle de un programa podés ver:

- Ciudadanos inscriptos
- Estado de cada inscripción
- Historial de actividades

### Derivaciones entre programas

Cuando un ciudadano necesita ser atendido por otro dispositivo o programa:

1. Desde el legajo, ir a **Derivaciones**
2. Seleccionar el programa destino
3. Agregar una nota de derivación
4. Confirmar

El sistema notifica automáticamente al dispositivo receptor.

## Permisos requeridos

La creación y edición de programas requiere rol de administrador institucional.
