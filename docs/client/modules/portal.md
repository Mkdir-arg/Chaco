# Módulo: Portal ciudadano

## ¿Qué es?

El portal ciudadano es la interfaz de acceso self-service para los ciudadanos. Permite consultar información personal, programas activos y realizar consultas sin necesidad de asistencia presencial.

## Acceso para ciudadanos

Los ciudadanos acceden desde la URL pública del portal (distinta al backoffice del staff).

### Registro de ciudadano

1. Ingresar a la URL del portal
2. Hacer clic en **Registrarse**
3. Completar DNI y datos de contacto
4. Verificar el correo electrónico recibido

### Funcionalidades disponibles para el ciudadano

- **Mi perfil** — ver y actualizar datos personales
- **Mis programas** — ver programas en los que está inscripto
- **Mis actividades** — historial de actividades registradas
- **Mis consultas** — enviar y seguir consultas al equipo
- **Trámites** — consultar el estado de trámites en curso

## Gestión desde el backoffice

El staff puede gestionar los accesos del portal desde **Portal → Usuarios ciudadanos**:

- Activar o desactivar cuentas
- Resetear contraseñas
- Ver el historial de accesos
