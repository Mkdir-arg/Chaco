# Módulo: Legajos

## ¿Qué es?

El módulo de Legajos permite registrar, consultar y gestionar la información de ciudadanos que interactúan con la institución.

## Funcionalidades principales

### Buscar un ciudadano

1. Ingresá a **Legajos** desde el menú lateral
2. Usá la barra de búsqueda por nombre, DNI o número de legajo
3. Hacé clic en el resultado para ver el detalle

### Registrar un ciudadano nuevo

1. Hacé clic en **Nuevo ciudadano**
2. Completá los datos del formulario
3. Podés buscar por DNI para autocompletar datos desde RENAPER (si está habilitado)
4. Guardá el registro

### Ver el legajo completo

El detalle de un ciudadano incluye pestañas con:

- **Datos personales** — información básica y de contacto
- **Programas** — programas en los que está inscripto
- **Derivaciones** — historial de derivaciones entre dispositivos
- **Alertas** — alertas activas asociadas al ciudadano
- **Archivos** — documentación adjunta

### Derivar a un programa

1. Desde el legajo del ciudadano, ir a la pestaña **Programas**
2. Hacer clic en **Derivar**
3. Seleccionar el programa y dispositivo destino
4. Confirmar la derivación

## Permisos requeridos

El acceso a este módulo depende del rol asignado. Consultá con el administrador de tu institución si no podés acceder a alguna funcionalidad.
