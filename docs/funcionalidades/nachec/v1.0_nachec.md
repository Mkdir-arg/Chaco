## Descripcion

ÑACHEC es el módulo específico para el programa de asistencia social familiar de SEDRONAR. Gestiona todo el ciclo de vida de un caso: desde la derivación inicial hasta el relevamiento sociofamiliar en campo, la evaluación de vulnerabilidad con scoring automático, el plan de intervención con prestaciones concretas, las tareas asignadas a territoriales y el seguimiento periódico. El scoring de vulnerabilidad evalúa 5 dimensiones: familia, ingresos, vivienda, salud/alimentación y riesgos.

## User Story

```
Como coordinador del programa ÑACHEC quiero gestionar casos de familias vulnerables para asignarlos a territoriales, registrar el relevamiento sociofamiliar, definir un plan de intervención con prestaciones concretas y hacer seguimiento territorial del impacto.
```

## Notas técnicas

- El scoring de vulnerabilidad se calcula en `RelevamientoNachec.calcular_scoring()` y devuelve score (0-100), categoría (BAJO/MEDIO/ALTO) y desglose por dimensión.
- El estado del caso se recalcula automáticamente según las prestaciones activas via `CasoNachec.recalcular_estado_por_prestaciones()`.
- `EvaluacionVulnerabilidad.override_categoria` permite al evaluador divergir de la categoría calculada, con justificación obligatoria.
- Las tareas tienen 5 tipos: RELEVAMIENTO, AMPLIACION, ENTREGA, SEGUIMIENTO, INSCRIPCION, VALIDACION.
- El relevamiento incluye geolocalización (`latitud`, `longitud`) para seguimiento territorial.
- Las prestaciones tienen confirmación de entrega con archivo de firma y foto.
